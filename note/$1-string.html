# 字符串

# 字符串的基本概念

字符串（英语：string），是由零个或多个字符组成的有限序列。一般记为一般记为 s = a1a2...an。它是编程语言中表示文本的数据类型。

# 字符串的基本操作

字符串的访问操作与数组类似，都是可以通过下标来访问，下面介绍一些常见的基础操作。
